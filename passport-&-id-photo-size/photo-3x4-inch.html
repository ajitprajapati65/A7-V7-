About 3 x 4 Inch Photos

  • Perfect Aspect Ratio: A 3x4 inch photo uses a 3:4 aspect ratio. Our tool securely locks the crop box so you never mess up the proportions.
  • Print Quality (DPI): To print a photo that looks sharp on paper, it needs to be 300 DPI. We automatically convert 3x4 inches into 900x1200 pixels, ensuring professional studio-grade quality.
  • Common Uses: This size is commonly used for larger identification cards, portfolio headshots, and specific international document submissions.
  • No Blurry Edges: Use the "High Quality (600 DPI)" option if you plan to print the image using a premium photo printer. It provides double the pixel density (1800x2400 px).
