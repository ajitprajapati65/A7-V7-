Standard 3.5 x 4.5 cm Guidelines

  • Head Proportion: Your face (from bottom of chin to top of hair) should cover about 70% to 80% of the photo area. Use the crop box to center your face perfectly.
  • Resolution (Pixels): Online forms don't read "cm", they read pixels. At standard 300 DPI, 3.5x4.5cm translates exactly to 413 x 531 pixels. Our tool locks this automatically.
  • File Size (KB): Portals like SSC, UPSC, and IBPS usually require the photo size to be between 20KB and 50KB. Use the "Image Quality" slider above to reduce the file size before downloading.
  • Background: A plain white or off-white background is universally accepted. If you uploaded a PNG with no background, select the White circle to auto-fill it.
